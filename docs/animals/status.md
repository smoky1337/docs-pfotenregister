# Tierstatus und Notizen

Einen expliziten Status-Schalter gibt es nicht; Status wird als Feld (z. B. aktiv/inaktiv) im Tieränderungsformular gepflegt. Notizen haben einen eigenen Schnell-Button.

## Status ändern
- Tier bearbeiten → Status-Feld anpassen (wenn sichtbar) → Speichern. Deaktivierte Tiere erscheinen nicht mehr in Ausgaben.

## Notizen bearbeiten
- Tierkarte → **Notizen bearbeiten** → Text speichern.

## Hinweise
- Sichtbarkeit/ Editierbarkeit des Status-Felds hängt von den Feldeinstellungen ab.
- Historische Ausgaben bleiben dem Tier zugeordnet, auch wenn der Status „inaktiv“ gesetzt wird.
