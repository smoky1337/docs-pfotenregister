# Tier registrieren

Tiere werden immer zu einem Gast angelegt. Sicht-/Pflichtfelder folgen den Feldeinstellungen.

## Aufruf & Rollen
- Gastdetail → **Tier hinzufügen** (`/animals/register?guest_id=<gast>`)
- Rollen: Admin, Editor

## Ablauf
1. Formularfelder ausfüllen (Name, Art, Rasse usw. – je nach Sichtbarkeit). 
2. Speichern → Tier erscheint im Gastprofil.

## Hinweise
- Felder, die nicht global sichtbar sind, fehlen im Formular.
- Pflichtfelder werden in den Feldeinstellungen gesteuert.
- Nach dem Speichern kannst du direkt zur Gastansicht zurückkehren; Tiere können später bearbeitet werden.
