# Gast aktivieren/deaktivieren

Der Status steuert, ob der Gast in Ausgaben berücksichtigt wird. Umschalten erfolgt aus der Liste (Modal) oder über Aktionen im Detail.

## Rollen & Aufruf
- Status ändern: nur Admin.
- Liste → Dropdown „Löschen/Deaktivieren“ (aktive) bzw. „Aktivieren/Löschen“ (inaktive) → Modal.

## Ablauf
- **Deaktivieren**: Modal bestätigen → Gast bleibt erhalten, erscheint nicht mehr bei Ausgaben.
- **Aktivieren**: Modal bei inaktiven Gästen → Gast wird wieder angezeigt.

## Hinweise
- Bei vorhandenen Zahlungen kann der Gast nicht gelöscht werden; Statuswechsel bleibt möglich.
- Aktivieren/Deaktivieren wird im Changelog vermerkt.
