# Erste Schritte

## 1. Anmelden
1. Browser öffnen (Chrome/Firefox empfohlen).
2. Instanz-URL eingeben (z. B. `verein.pfotenregister.com`).
3. Benutzername/E-Mail + Passwort eingeben → **Anmelden**. 

## 2. Startseite nutzen (Gast aufrufen)
- Barcode der Gästekarte scannen **oder** 6-stelligen Gast-Code eingeben.
- Alternativ Name tippen → Vorschläge (aus Liste) auswählen → führt direkt zur Gastdetailseite.
- Rollenabhängige Schnellaktionen: „Neuen Gast erfassen“, „Alle Gäste anzeigen“.

## 3. Navigationsleiste oben verstehen
- Links: Gäste, Tiere, Ausgaben, Zahlungen, Hinweise.
- Rechts: Adminbereich, Profil, Hilfe Log-Out.

## 4. Erste Aufgaben
- **Neu starten**: Gast erfassen → direkt Tier anlegen → Gästekarte drucken.
- **Bestehenden Gast bedienen**: Barcode scannen → Futter ausgeben → optional Zahlung buchen.

## 5. Hilfe
- Passwort vergessen: Admin fragen.
