# Workflow: Futterausgabe durchführen

So dokumentierst du eine Ausgabe vom Aufruf des Gastes bis zur Quittung.

## 1) Gast aufrufen
- QR-Code scannen oder in der Gästeliste suchen → Gastprofil öffnet sich.
- Prüfe Status (aktiv?) und Hinweise

## 2) Hinweise prüfen
- Tab **Futterausgaben**: Futterhinweise, Tags, letzte Ausgabe ansehen.
- Bei Warnungen (Sperrfrist) entscheiden, ob du trotzdem ausgeben darfst.

## 3) Ausgabe erfassen
1. **Futter ausgeben** klicken.
2. Tiere bestätigen/auswählen.
3. Menge/Notiz eintragen, Tags wählen (z. B. „Winterpaket“).
4. Speichern.

## 4) Optional: Zahlung oder Quittung
- Falls ein Beitrag fällig ist: Tab **Zahlungen** → Zahlung erstellen oder Vormerkung.
- Quittung drucken, wenn aktiviert.

## 5) Abschluss
- Prüfe, ob die Historie den neuen Eintrag zeigt.
- Karte ggf. erneut drucken, wenn Tiere oder Hinweise geändert wurden.

## Tipps
- Nutze kurze Notizen („+1 Nass, Warnung akzeptiert“), damit andere Helfer:innen wissen, was passiert ist.
