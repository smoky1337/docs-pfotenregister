# Workflow: Zahlung erfassen und korrigieren

Dieses Beispiel zeigt den Weg von einer offenen Vormerkung bis zur Bestätigung oder Gegenbuchung.

## 1) Vormerkung anlegen
- Gast → Tab **Zahlungen** → **Vormerkung**.
- Betrag, Fälligkeit und Kommentar erfassen („10 € nächste Woche“).
- Vormerkung erscheint als offen.

## 2) Zahlungseingang verbuchen
- Vormerkung öffnen → **Bestätigen**.
- Zahlungsart wählen, Kommentar ergänzen („bar am 12.04.“) → Speichern.
- Vormerkung wird zur Zahlung.

## 3) Fehler korrigieren
- Falschen Betrag gebucht? Zahlung öffnen → **Gegenbuchung** erstellen → korrekten Betrag neu buchen.
- Falscher Gast? Gegenbuchen, dann Zahlung beim richtigen Gast erfassen.

## 4) Auswerten
- Bericht „Zahlungsjournal“ für den Zeitraum exportieren.
- Offene Vormerkungen auf dem Dashboard oder im Bericht filtern.

## Tipps
- Klare Kommentare („Ratenzahlung 1/3“) helfen bei Nachverfolgung.
- Nutze Vormerkungen, wenn Gäste erst am Monatsende zahlen – so bleibt der Überblick erhalten.
