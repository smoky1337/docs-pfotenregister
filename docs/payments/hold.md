# Zahlung offen erfassen

Eine „offene“ Zahlung entspricht einer Buchung ohne Haken bei „bezahlt“. Sie kann später markiert oder gelöscht werden.

## Aufruf & Rollen
- Gastdetail → Zahlungen → Formular „Zahlung erfassen“
- Rollen: Admin, Editor

## Schritte
1. Beträge eintragen.
2. Checkbox „bezahlt“ **nicht** anhaken.
3. Kommentar hinzufügen (z. B. „zahlt nächste Woche“).
4. Speichern. Zahlung erscheint als offen.

## Hinweise
- Offene Zahlungen lassen sich später über „Als bezahlt markieren“ abschließen.
- Offene Zahlungen können gelöscht werden, solange sie nicht bezahlt sind.
