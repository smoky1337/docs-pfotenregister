# Admin: Foodtags verwalten

Foodtags sind frei definierbare Labels mit Farbe. Sie erscheinen bei Tieren und Ausgaben, wenn das Tagsystem aktiv ist.

## Voraussetzungen
- Tagsystem in den allgemeinen Einstellungen auf **Aktiv** gestellt.

## Arbeiten mit Tags
1. Admin → Einstellungen → Tab **Foodtags**.
2. **Vorhandene Tags bearbeiten**: Namen oder Farbe in der Tabelle anpassen → Speichern.
3. **Neue Tags anlegen**: *Neuen Tag hinzufügen* klicken → Name/Farbe eintragen → Speichern.
4. **Tags löschen**: Mülleimer klicken → Zeile wird beim Speichern entfernt.

## Hinweise
- Doppelte Namen werden beim Speichern abgefangen; wähle eindeutige Bezeichnungen.
- Tags können sowohl Tieren als auch Futterausgaben zugewiesen werden.
- Bei deaktiviertem Tagsystem verschwindet der Tab und die Tag-Funktionen im Frontend.
