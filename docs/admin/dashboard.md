# Admin: Dashboard

Das Dashboard bündelt Kennzahlen und Direktzugriffe für Admins.

## Was du siehst
- **Kennzahlen**: Gäste gesamt, aktive Gäste, Gäste mit Futter in den letzten 30 Tagen, Tiere gesamt.
- **Diagramme**: Tierverteilung (Torte), Top-10-Gäste nach Besuchen (Balken, klickbar), Zahlungsverlauf der letzten 30 Tage (Linie für Futter/Zubehör).
- **Foodtag-Tabelle**: Tag → Anzahl zugeordneter Tiere (nur sichtbar, wenn Tagsystem aktiv).
- **Schnellzugriffe**: Nutzerverwaltung, Nutzer anlegen, Gästekarten drucken, Einstellungen, Daten exportieren/importieren, Zahlungsexport.

## Typische Workflows
- **Gast mit vielen Besuchen prüfen**: Im Top-10-Diagramm auf eine Säule klicken → Gastdetail öffnet sich → Status oder Hinweise anpassen.
- **Zahlungen exportieren**: Button *Zahlungen exportieren* → Zeitraum im Modal wählen (7/30/90/180 Tage oder seit 01.01.) → Export starten → Bericht im Browser prüfen oder als PDF drucken.
- **Daten sichern**: *Daten exportieren* → Tabellen/Spalten wählen → XLSX herunterladen.
- **Import vorbereiten**: *Daten importieren* → Excel-Vorlage laden → Datei hochladen → Vorschau kontrollieren → Import bestätigen.

## Hinweise
- Diagramme nutzen die letzten Einträge bzw. 30 Tage. Zahlungsdiagramm erscheint nur, wenn Zahlungen aktiviert sind.
- Foodtag-Tabelle erscheint nur bei aktiviertem Tagsystem.
- Gästekarten-Druck erlaubt Sammelauswahl; Druckdatum wird gespeichert.
