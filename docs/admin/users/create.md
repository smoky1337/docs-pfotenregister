# Admin: Nutzer anlegen

Neue Zugänge erstellst du über die Admin-Nutzerverwaltung.

## Aufruf
- Dashboard → **Benutzerverwaltung** → **Neuer Nutzer** (`/admin/users/register`)

## Felder
- **Benutzername**: Login-Name (muss eindeutig sein).
- **Rolle**: Admin oder Editor oder Nutzer, haben absteigend viele Rechte.
- **Passwort**: Startpasswort.
- **Name**: Anzeigename/Realname (optional).

## Ablauf
1. Formular ausfüllen (Benutzername, Rolle, Passwort, Name).
2. **Speichern**. Bei doppeltem Benutzername erscheint ein Hinweis.
3. Der Account ist sofort aktiv.

## Hinweise
- Rollen wirken sofort und steuern Zugriff auf Funktionen.
- Passwörter werden verschlüsselt gespeichert.
