# Tags verwalten (Futter & Tiere)

Tags markieren besondere Anforderungen oder Pakettypen. Sie erscheinen in Futterhinweisen und Ausgaben.

## Zielgruppe & Voraussetzungen
- Anlegen/Bearbeiten: Admin (über Einstellungen → Tags)
- Anwenden: Admin, Editor

## Tags anwenden
1. Beim Tier oder Futterhinweis das Feld **Tags** öffnen.
2. Bestehende Tags suchen oder auswählen.
3. Speichern. Der Tag erscheint sichtbar auf der Detailseite und in der Ausgabe.

## Typische Workflows
- **Allergien kennzeichnen**  
  Tag „Allergie“ + Kommentar im Hinweis setzen; in der Ausgabe wird der Warnhinweis sichtbar.

- **Pakettypen markieren**  
  Tags wie „Katzenpaket“, „Hundefutter groß“, „Winterpaket“ bei der Ausgabe auswählen.

- **Temporäre Aktionen**  
  Saisonale Tags auswählen und nach Ende wieder entfernen.

## Hinweise
- Welche Tags verfügbar sind, steuert der Admin in den Tag-Einstellungen.
- Tags beeinflussen keine Mengen, helfen aber beim Filtern und Auswerten.
