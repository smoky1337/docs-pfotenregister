# Futter ausgeben

Futtereinträge werden pro Gast angelegt. Optional kannst du Tags hinzufügen und gleichzeitig eine Zahlung erfassen (wenn aktiviert).

## Aufruf & Rollen
- Gastdetail → Bereich **Futterausgaben** → Formular „Futter ausgeben“
- Rollen: Admin, Editor

## Schritte
1. Notiz eintragen (frei).
2. Optional Beträge: `futter_betrag`, `zubehoer_betrag`; Kommentar für Zahlung.
3. Tags auswählen
4. Speichern → Eintrag wird für „heute“ angelegt.

## Hinweise
- Beträge werden nur verbucht, wenn das Zahlungsmodul aktiv ist. Andernfalls wird nur die Ausgabe gespeichert.
- Tags hängen am Futtereintrag und stammen aus den Foodtags.
- Sperrfristen/Erinnerungen werden nicht automatisch im Formular blockiert; beachte Hinweise im Gastprofil.
