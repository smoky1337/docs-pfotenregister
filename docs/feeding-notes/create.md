# Futterhinweis / Notiz erfassen

Im System werden Hinweise als Kommentar pro Futtereintrag oder als Tier-Notiz geführt. Es gibt keine separate Hinweisverwaltung – trage Hinweise direkt in den Feldern ein.

## Optionen
- **Notiz im Futtereintrag**: Beim Ausgeben im Feld „Notiz“ festhalten (z. B. Menge, Besonderheiten). Wird in der Historie angezeigt.
- **Tiernotiz**: In der Tierkarte „Notizen bearbeiten“ → speichert dauerhaft beim Tier.

## Hinweise
- Kurze, präzise Texte helfen an der Ausgabe.
- Tags (Foodtags) können ergänzend gesetzt werden, um Anforderungen sichtbar zu machen.
