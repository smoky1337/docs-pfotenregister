# Futtereintrag bearbeiten oder löschen

Ein Futtereintrag enthält Datum und Notiz. Du kannst Datum/Notiz anpassen oder den Eintrag löschen.

## Bearbeiten
- Aufruf: Gastdetail → Futterausgabe-Eintrag → **Bearbeiten** (`/feed_entry/<id>/edit`)
- Rollen: Admin, Editor
- Ändere Datum (`futtertermin`) und Notiz → Speichern.

## Löschen
- Aufruf: Gastdetail → Futterausgabe-Eintrag → **Löschen** (`/feed_entry/<id>/delete`)
- Rollen: Admin, Editor
- Entfernt den Eintrag endgültig.

## Hinweise
- Tags am Eintrag bleiben nur erhalten, wenn das Formular sie unterstützt; beim Bearbeiten werden aktuell Datum/Notiz geändert.
- Bei Fehlern lieber korrigieren als löschen, wenn Historie benötigt wird.
